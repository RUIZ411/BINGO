빙고 방 선택

방을 생성하려면 관리자 계정으로 인증하고, 생성된 5자리 입장 코드로 방에 접속하세요.

방 코드는 접속 권한을 구분하기 위한 간편 코드입니다. 코드가 공유되면 같은 방을 조작할 수 있어요.

HOW TO USE

사용 방법

방을 먼저 만들고, 입장 코드로 접속한 뒤 빙고판을 생성해서 사용하면 됩니다.

1 방 생성

원하는 방에서 방 생성을 누르고, 관리자 아이디와 비밀번호로 인증합니다.

2 입장 코드 공유

방이 생성되면 5자리 입장 코드가 발급됩니다. 이 코드를 함께 사용할 사람에게 공유하세요.

3 빙고판 조작

입장 후 빙고판 생성, 체크/복구, 현상금, 메모장, 칸 수정 모드를 사용할 수 있습니다.

4 송출용 사용

각 방의 송출용 버튼을 누르면 OBS에 넣기 좋은 전용 화면으로 열립니다.
