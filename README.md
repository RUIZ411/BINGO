# SUWEET BINGO V1 - 모든 빙고 현상금 대상 지원 버전

## 이번 버전 수정점

### 현상금 기능 확장

기존에는 현상금 대상 입력칸이 숫자만 받을 수 있었지만, 이제 모든 빙고 타입에서 사용할 수 있습니다.

- 숫자 빙고: `25`, `77`, `100` 등
- 알파벳 빙고: `A`, `B`, `Z` 등
- 리셋 빙고: `A`, `B`, `Z` 등
- 미션 빙고: 칸에 적힌 미션명 그대로 입력

관리자 메뉴에서 대상 칸과 현상금 개수를 입력하면, 빙고판에서 같은 내용의 칸이 빨간색으로 표시되고 아래에 현상금 개수가 표시됩니다.

예시:

```txt
대상 칸: A
개수: 300
```

또는

```txt
대상 칸: 수류탄 5킬
개수: 500
```

## 사용 방법

1. 관리자 로그인
2. 우측 관리자 메뉴의 `현상금 설정` 섹션으로 이동
3. `대상 칸`에 빙고판 칸 내용 입력
4. `개수`에 현상금 개수 입력
5. `추가 / 수정` 클릭

대상 칸이 현재 빙고판에 없으면 확인창이 뜹니다. 그래도 등록하면 나중에 같은 내용으로 칸을 수정했을 때 자동으로 현상금 표시가 적용됩니다.

## 유지된 기능

- 칸 수정 모드
- 메모장 팝업
- 숫자 자릿수별 큰 폰트
- 알파벳/리셋 글자 확대
- 숫자 랜덤 추첨
- 리셋 빙고 메뉴판
- 송출용 어두운 배경 + 비율 조절
- `SUWEET BINGO` 브라우저 타이틀
- `송출용` 버튼명
- Firebase DB URL 적용
- `@suweet.com` 자동 로그인
- 세션 로그인 방식
