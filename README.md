# SUWEET BINGO V1

Firebase Realtime Database 기반 방송용 빙고 사이트입니다.

## 포함 기능

- 메인 화면: 좌측 빙고판 / 우측 관리자 패널
- 송출용 화면: 어두운 배경 + 중앙 송출 박스 + 비율 조절
- 숫자 빙고: 5x5 / 7x7 / 10x10
- 미션 빙고: 5x5
- 알파벳 빙고: 5x5, A~Z 중 25개 랜덤 배치
- 리셋 빙고: 5x5, A~Z 중 25개 랜덤 배치
- 숫자 랜덤 추첨: 1~100 중복 없이 추첨, 빙고판에 있으면 자동 지움
- 현상금 숫자: 관리자 메뉴에 입력한 숫자는 빙고판에서 빨간색 표시
- 리셋 빙고 메뉴판: 한개/한줄/전체 개수 설정
- 메모장 팝업
- 칸 수정 모드: 관리자 메뉴에서 ON 후 빙고칸 클릭으로 내용 직접 수정
- 빙고 수 / 치킨 수 실시간 표시
- Firebase 실시간 공유
- `@suweet.com` 자동 보완 로그인
- 세션 로그인 방식

## 화면 주소

```txt
메인 화면: index.html?view=public
송출용: index.html?view=obs
```

송출용 비율 직접 지정:

```txt
index.html?view=obs&scale=1
index.html?view=obs&scale=0.8
index.html?view=obs&scale=0.67
index.html?view=obs&scale=0.5
```

## 칸 수정 모드 사용법

1. 관리자 로그인
2. 우측 관리자 패널의 `칸 수정 모드`에서 `수정 모드 켜기` 클릭
3. 왼쪽 빙고판에서 수정할 칸 클릭
4. 새 내용 입력 후 저장
5. 수정이 끝나면 `수정 모드 끄기`

입력 제한:

- 숫자 빙고: 1~100 숫자만 가능
- 알파벳/리셋 빙고: A~Z 한 글자만 가능
- 미션 빙고: 자유 텍스트 가능

숫자/알파벳/리셋 빙고에서 이미 같은 값이 있으면 경고 후 진행 여부를 선택할 수 있습니다.
