# SUWEET BINGO V1 - Reset Menu Layout

## 이번 버전 추가 사항

- 리셋 빙고 선택 시 송출/메인 빙고판 아래 영역이 전용 메뉴판 형태로 변경됩니다.
- 리셋 빙고에서는 기존 가로형 `BINGO / 🍗` 카운터 대신 다음 구조가 표시됩니다.

```text
왼쪽: BINGO 수 / 🍗 수 세로 나열
오른쪽: 한개 N개 / 한줄 N개 / 전체 N개 메뉴판
```

- 관리자 패널에 `리셋 빙고 메뉴판` 설정 섹션이 추가되었습니다.
  - 한개
  - 한줄
  - 전체
- 위 3개 값은 Firebase에 저장되어 공개 화면과 송출용 화면에 실시간 반영됩니다.
- 일반 숫자/미션/알파벳 빙고는 기존 디자인을 유지합니다.

## 기존 유지 기능

- Firebase Realtime Database 연결 적용
- @suweet.com 자동 로그인
- 세션 로그인 방식
- 숫자 빙고 5x5 / 7x7 / 10x10
- 미션 / 알파벳 / 리셋 빙고 5x5 고정
- 알파벳 / 리셋 빙고 A~Z 중 25개 랜덤 배치
- 숫자 랜덤 추첨
- OBS 어두운 배경 + 비율 조절

## 이번 수정
- 숫자 빙고일 때 숫자 폰트 크기를 더 크게 조정했습니다.
- 미션/알파벳/리셋 빙고의 글자 크기는 기존 가독성 설정을 유지합니다.


## 숫자 자릿수별 폰트 확대

숫자 빙고에서 1자리/2자리/3자리 숫자의 폰트 크기를 다르게 적용해 칸을 더 크게 채우도록 수정했습니다.


## 현상금 숫자 기능

관리자 패널의 `현상금 숫자` 입력칸에 숫자를 입력하고 `현상금 적용`을 누르면, 현재 숫자 빙고판에서 같은 숫자의 글씨가 빨간색으로 표시됩니다.

- 쉼표, 공백, 줄바꿈 모두 구분자로 사용 가능
- 1~100 사이 숫자만 적용
- 중복 입력은 자동 제거
- 송출용 화면에도 같은 빨간색 표시가 반영됨

## 메모장 팝업 기능

상단 메뉴에 `📝 메모장` 버튼이 추가되었습니다.

- 메인 화면에서 버튼 클릭 시 팝업 메모장이 열립니다.
- 관리자 로그인 상태에서만 저장/비우기가 가능합니다.
- 메모 내용은 Firebase `memoText` 필드에 저장되어 같은 사이트를 보는 화면에 동기화됩니다.
- 송출용 화면에서는 메모장 버튼과 팝업이 표시되지 않습니다.
- `ESC` 키 또는 바깥 배경 클릭으로 닫을 수 있습니다.
