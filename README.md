# SUWEET BINGO V1 - 현상금 개수 표시 버전

## 추가된 기능

### 현상금 숫자 + 개수 표시

관리자 메뉴의 현상금 숫자에서 숫자와 개수를 함께 등록할 수 있습니다.

예시:
- 숫자: 25
- 개수: 300

등록 후 숫자 빙고판에 25가 있으면 칸 안에 아래 구조로 표시됩니다.

```txt
25
300
```

- 목표 숫자 폰트 크기는 기존 숫자 빙고 크기를 최대한 유지합니다.
- 현상금 개수는 숫자 아래에 작지만 확인 가능한 크기로 표시됩니다.
- 현상금 숫자는 빨간색으로 표시됩니다.
- 메인 화면과 송출용 화면 모두 반영됩니다.
- Firebase에 저장되어 다른 화면에서도 실시간으로 동기화됩니다.

## 유지된 기능

- 칸 수정 모드
- 메모장 팝업
- 현상금 숫자 기능 확장
- 숫자 자릿수별 큰 폰트
- 숫자 랜덤 추첨
- 리셋 빙고 메뉴판
- 송출용 어두운 배경 + 비율 조절
- `SUWEET BINGO` 타이틀
- `송출용` 버튼명
- Firebase DB URL 적용
- `@suweet.com` 자동 로그인
- 세션 로그인 방식
