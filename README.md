# SUWEET Bingo + Number Battle Rooms

이번 버전은 기존 빙고방 4개를 유지하면서, 숫자빙고 전용 대결방 4개를 추가한 버전입니다.

## 포함 기능

### 기존 빙고방 유지
- 1번 방 / 2번 방 / 3번 방 / 수힛방 유지
- 관리자 패널 타입별 정리 유지
- 숫자 랜덤 추첨 제거 유지
- 리셋빙고 줄/전체 체크 해제 유지
- 숫자빙고 10×10은 1~100 순서 배치 유지

### 신규 대결방
- 대결방 1~4 추가
- 대결방 입장 코드 / 관리자 인증 구조
- 숫자빙고 전용 대결 모드
- 참가 팀 수: 2팀 또는 3팀
- 빙고 크기: 5×5 / 7×7 / 10×10
- 5×5, 7×7: 팀별 서로 다른 랜덤 숫자판
- 10×10: 모든 팀 1~100 순서판
- 팀별 체크 상태 개별 저장
- 팀별 체크 수 / 진행률 표시
- 선택 팀 상단에 상대 팀 진행률 표시
- 팀별 BINGO 수 / 치킨 수 표시
- 송출용 화면에서 팀 탭 제공
- 팀별 직접 송출 URL 지원

## Firebase Rules

이번 버전은 `battleRooms` 경로가 추가됩니다.

```json
{
  "rules": {
    "bingoRooms": {
      ".read": true,
      ".write": true
    },
    "battleRooms": {
      ".read": true,
      ".write": true
    }
  }
}
```

## 대결방 사용 방법

1. 홈 화면에서 대결방 1~4 중 원하는 방의 `방 생성`을 누릅니다.
2. 관리자 계정으로 인증합니다.
3. 생성된 5자리 입장 코드로 대결방에 입장합니다.
4. 대결 제목, 참가 팀 수, 빙고 크기를 설정합니다.
5. `대결판 새로 생성`을 누르면 팀별 숫자판이 생성됩니다.
6. 팀 탭을 눌러 각 팀의 빙고판을 체크합니다.
7. 송출용 버튼을 누르면 OBS용 화면으로 볼 수 있습니다.

## 대결방 송출 URL

- 대결방 전체 송출용: `?battleRoom=battle1&view=battleObs`
- 팀 1 직접 송출용: `?battleRoom=battle1&view=battleObs&team=0`
- 팀 2 직접 송출용: `?battleRoom=battle1&view=battleObs&team=1`
- 팀 3 직접 송출용: `?battleRoom=battle1&view=battleObs&team=2`


## 핫픽스
- 기존 빙고방 public 화면의 우측 관리자 패널/빙고판 렌더링 레이아웃을 강제 복구했습니다.
- battleRooms Firebase Rules가 아직 반영되지 않은 상태에서도 기존 빙고방 화면이 멈추지 않도록 대결방 목록 수신을 홈 화면으로 분리했습니다.


## 홈 화면/방 목록 안정화 핫픽스

- 사이트 첫 접속 시 Firebase 수신 전에 기본 방 카드가 먼저 렌더링되도록 수정했습니다.
- `bingoRooms` 또는 `battleRooms` 읽기 실패 시에도 방 목록 화면이 빈 화면으로 멈추지 않도록 방 목록 렌더링을 분리했습니다.
- 상단 첫 버튼을 `방 선택`으로 변경하여 언제든 홈 방 목록으로 이동할 수 있게 했습니다.
- 기존 빙고방 4개와 대결방 4개 구조는 유지됩니다.
